# QYL Amazon Japan Calculator

Simple offline HTML calculator for China-to-Japan Amazon pricing.

- Activity price formula (CNY):
  ((purchase + shipping) + ((purchase + shipping) / 50%) * 35%) / 85%
- Daily price (JPY): activity * 1.2
- Retail reference (JPY): activity * 1.6
- Default FX: 23 JPY per 1 CNY

Open `QYL_amazon_japan_calculator.html` directly in a browser.
